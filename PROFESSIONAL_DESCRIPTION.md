The **KII Product Tag Generator** is a premium, cross-platform desktop application designed for manufacturing and industrial environments. This professional-grade tool enables businesses to create high-quality product tags with embedded QR codes for inventory management, quality control, and product identification.

## üéØ **Key Features & Capabilities**

### **Professional Tag Generation**
- Create manufacturing-quality product tags with precise specifications
- Generate high-resolution QR codes (600 DPI) with embedded company branding
- Support for multiple layout formats: Vertical and Horizontal professional layouts
- Real-time preview with pixel-perfect alignment and zoom functionality

### **Advanced Customization**
- **Three Professional Themes**: Ocean Azure (Light), Midnight Blue (Dark), and Industrial
- Customizable tag dimensions with configurable DPI settings
- Support for company logo embedding in QR codes
- Flexible color schemes for different industrial environments

### **Comprehensive Data Management**
- Product information fields: Name, Part Number, QC Status, Manufacturing Origin
- Catalog URL integration for detailed product information access
- Quality Control status indicators: Approved, Not Approved, Prototype
- Persistent user settings and preferences

### **Professional Export Capabilities**
- **Multiple Export Formats**: PNG (transparent), JPG (compressed), PDF (print-ready)
- **High-Resolution Output**: 600 DPI for professional printing standards
- **Batch Processing**: CSV import for large-scale print runs
- Configurable output dimensions and quality settings
